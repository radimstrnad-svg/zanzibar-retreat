# Zanzibar — jógový retreat

Jednostránkový interaktivní web pozvánky na jógový retreat na Zanzibaru (2.–11. 10. 2026), pořádá YOGASEE.CZ.

## Obsah
- `index.html` — celá stránka (HTML + CSS + JS v jednom souboru)
- `img1–3.jpg` — fotky resortu White Paradise Zanzibar
- `cell_*.jpg` — fotky pokojů a bungalovů
- `stepanka.png` / `misa.png` — fotky lektorek

## Funkce
- Scroll-reveal animace, hover efekty, plynulý scroll
- Plně responzivní (desktop / tablet / mobil)
- Klikací telefon a e-mail

## Nasazení
Statická stránka. Live: https://zanzibar-retreat.vercel.app
